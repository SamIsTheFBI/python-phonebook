## PhoneBook

A simple phone book app written in Python.

```
git clone https://github.com/SamIsTheFBI/python-phonebook
cd python-phonebook
pip install -r requirements.txt
flask --app app.py run
```

Visit http://localhost:5000/ on your browser.
